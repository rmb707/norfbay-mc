# Playing on Norfbay 🎮

You need **Minecraft: Java Edition** (a Microsoft account that owns the game).

## The easy way (once the launcher is out)
1. Go to **play.norfbay.com** and sign in with Google (Rob adds your email to the list).
2. **Download the Norfbay Launcher** and run it.
3. Sign in with your **Microsoft** (Minecraft) account.
4. Pick **Cobblemon** or **Vanilla+** → hit **Connect**.
   It installs the right Minecraft version + mods and drops you straight into the server.
   (First time downloads a bit; after that it's instant.)

## The interim way (right now, before the launcher ships)
Use the free **Modrinth App** or **Prism Launcher**:
1. Install the [Modrinth App](https://modrinth.com/app).
2. Import the pack file Rob shares (`norfbay-cobblemon.mrpack` or `norfbay-vanilla.mrpack`).
3. Hit **Play**, then in Multiplayer connect to:
   - Cobblemon → `cobblemon.norfbay.com` (or `minecraft.norfbay.com`)
   - Vanilla+ → `vanilla.norfbay.com`
4. Tell Rob your Minecraft username so he can whitelist you.

## Notes
- Both servers are Minecraft **1.21.1 (Fabric)** — the pack sets that up for you.
- If you get "You are not white-listed", ping Rob with your exact Minecraft name.
- The launcher may show a Windows SmartScreen warning (it's unsigned) — "More info → Run anyway".
